A high-tech, modern personal portfolio website built with **React.js**. This project features a glassmorphism UI, neon aesthetics, and is optimized for showcasing Full Stack development skills and academic projects.

## ✨ Features
- **Modern UI:** Dark mode with glassmorphism effects and glowing neon accents.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens.
- **Component-Based Architecture:** Clean and modular React code.
- **Smooth Navigation:** Seamless transitions using `React Router` (HashRouter).
- **Dynamic Projects Page:** Specialized cards highlighting tech stacks and GitHub integration.

---

## 🛠️ Tech Stack
- **Frontend:** React.js, JavaScript (ES6+), CSS3 (Flexbox/Grid)
- **Routing:** React Router (HashRouter for GitHub Pages compatibility)
- **Styling:** Custom CSS with Glassmorphism and Neon Glow animations
- **Deployment:** GitHub Pages
